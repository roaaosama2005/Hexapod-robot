About the Project

The Hexapod Robot Project is a six-legged research platform designed for robust locomotion, real-time vision and modular expansion. Each leg has multiple actuated joints, enabling complex gaits and stable traversal of rough terrains. The platform uses STM32-based control, ESP32-CAM for live streaming, and PCA9685 servo drivers for scalable actuation.

Key Features

Smart Gaits

Multi-gait system with tripod, ripple and wave gaits for different speeds and stability.

Live Streaming

ESP32-CAM streams live video to mobile devices for remote piloting and inspection.

Adaptive Balance

IMU and sensor feedback used for automatic posture compensation and stability control.

Modular Platform

Easily add sensors (LiDAR, ultrasonic), payloads or compute modules for research experiments.

Main Components

STM32F103C8T6 (Blue Pill)

Primary real-time controller for servo sequencing and motion planning.

ESP32-CAM

Onboard camera module for live video streaming and potential vision processing.

PCA9685 Servo Driver

16-channel PWM driver used to expand control of multiple high-torque servos.

HC-05 Bluetooth Module

Bluetooth interface for mobile control and debugging sessions.

How It Works

Motion Control

Inverse kinematics compute each leg's joint angles; trajectories are executed with PWM-synced servo control.

Perception

ESP32-CAM provides a live feed; additional sensors (IMU, ultrasonic) provide environment and orientation data.

Wireless Control

Control via mobile app or web UI — switching gaits, camera view and movement commands in real time.

AI Upgrades

Optional computer vision for object tracking and basic obstacle avoidance (future integration).
