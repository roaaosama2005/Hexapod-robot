About the Project

This Hexapod Robot is designed to move with insect-like grace using six individually actuated legs. Each leg has multiple joints, enabling complex motions. It’s controlled by a microcontroller (STM32F103C8T6 (Blue Pill)) and features wireless communication and live video streaming.

It is built for rugged terrains and stability, combining mechanical design with intelligent control algorithms to navigate uneven surfaces smoothly.
